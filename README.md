# Supply-Chain-Analytics-End-to-End
Overview

End-to-end supply chain analytics project built using Excel, SQL Server, and Power BI to monitor supplier performance, inventory health, and stockout risk.

This project transforms raw operational data into a structured data model and interactive dashboards that support procurement, planning, and operations decision-making.

Business Objective

Provide visibility into:

Supplier delivery performance

Inventory levels and stockout risk

Reorder planning and urgency

Warehouse risk exposure

Financial impact of inventory decisions

Tech Stack

Excel (raw operational data)

SQL Server (data cleaning & transformation)

Power BI (data modeling, DAX, dashboards)

Star schema modeling

Data Workflow

Raw data collected in Excel

Cleaned and standardized in SQL Server

Built fact and dimension tables

Modeled into a star schema

Imported into Power BI

DAX measures created for KPIs

Interactive dashboards developed
